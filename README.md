# Litty

Litty frontend written in Vue.js.

## Requirements

* Vue.js v2
* Node.js v12+

## Installation

Use [npm](https://npmjs.com/) to install modules to run Litty.

### npm
```bash
npm i
```

## How to run

### npm
```bash
npm run dev-fix
```

### yarn
```bash
yarn run dev-fix
```
